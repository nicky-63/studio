# **App Name**: Career Compass

## Core Features:

- User Profile Creation: Create and manage user profiles with personal details, academic background, and career interests captured during the signup process.
- Skill Assessment: Administer a dynamic questionnaire to assess skills, interests, and preferences across various engineering domains. The AI tool assesses the results to create an individualised skills profile. 
- Career Path Recommendation: Utilize AI to match user profiles with suitable career paths, suggesting roles based on assessed skills, interests, and industry preferences. 
- Skill Gap Analysis: Compare a student's current skills against the requirements for recommended career paths, identifying gaps and suggesting actions for skills to aquire.
- Personalized Roadmap Generation: Generate custom roadmaps including courses, certifications, side projects, and internships, tailored to bridge identified skill gaps and achieve career goals.
- Course Integration: Maintain and provide relevant course recommendations from a database or external APIs such as Coursera and Udemy. Filter, find, and recommend suitable courses using AI to filter course options, using semantic similarity between skills, keywords, course description, etc.
- Progress Tracking: Enable users to track progress on recommended courses, projects, and skills, visualizing growth and achievements through a personal dashboard.
- Informational pages: Provide Home, Features, Services, Career, Gallery and Contact Us pages, together with comprehensive login and signup screens

## Style Guidelines:

- Primary color: Deep teal (#008080) evokes professionalism, knowledge, and trust.
- Background color: Light teal (#E0F8F8), a muted version of the primary, offers a clean and unobtrusive backdrop.
- Accent color: Warm coral (#FF7F50), an analogous color, will highlight CTAs.
- Headline font: 'Space Grotesk' sans-serif, creates a modern, clean, technical feel; use 'Inter' for body text.
- Crisp, geometric icons related to engineering, skills, and career development.
- Clean and structured layout with clear sections for profile, assessment, recommendations, and progress tracking. Back buttons should be prominent for easy navigation.
- Subtle transitions and feedback animations on user interactions, such as skill assessments and progress updates.
